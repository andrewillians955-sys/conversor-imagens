LGPD e Proteção de Dados

Entenda como aplicamos a Lei Geral de Proteção de Dados (LGPD) neste site.

Princípios de privacidade

A Lei nº 13.709/2018, conhecida como Lei Geral de Proteção de Dados Pessoais (LGPD), define regras para o tratamento de dados pessoais no Brasil. Nosso compromisso é seguir os princípios de necessidade, finalidade, transparência e segurança, evitando a coleta de dados desnecessários e protegendo as informações que eventualmente sejam tratadas.

Tratamento de imagens e arquivos

Nosso conversor foi projetado para minimizar o tratamento de dados pessoais. As imagens WebP e HEIC que você escolhe converter são processadas localmente, diretamente no seu navegador, sem upload para servidores controlados por nós. Dessa forma, não coletamos nem armazenamos as imagens enviadas, reduzindo significativamente a exposição de dados sensíveis.

Dados de navegação e cookies

Eventualmente podem ser coletados dados de navegação anonimizados para fins estatísticos e de publicidade, como páginas visitadas, tipo de dispositivo e tempo de permanência. Esses dados não são utilizados para identificar individualmente os usuários, mas sim para compreender o uso do site e otimizar a exibição de anúncios, em especial por meio de plataformas como o Google AdSense.

O uso de cookies é detalhado na nossa Política de Cookies, e você pode gerenciar o consentimento por meio do banner exibido ao acessar o site.

Seus direitos como titular de dados

De acordo com a LGPD, você possui diversos direitos em relação aos seus dados pessoais, incluindo:

  • Confirmação da existência de tratamento de dados pessoais;
  • Acesso aos dados pessoais eventualmente tratados;
  • Correção de dados incompletos, inexatos ou desatualizados;
  • Anonimização, bloqueio ou eliminação de dados desnecessários ou excessivos;
  • Portabilidade dos dados a outro fornecedor de serviço (quando aplicável);
  • Informação sobre compartilhamento de dados com entidades públicas ou privadas;
  • Revogação do consentimento e informação sobre a possibilidade de não fornecê-lo;
  • Oposição ao tratamento realizado com fundamento em outras bases legais, em caso de descumprimento da lei.

Para exercer qualquer um desses direitos, entre em contato pelo e-mail contato@convertflash.com. Teremos o cuidado de analisar cada solicitação à luz da legislação aplicável.

Segurança e retenção de dados

Empregamos medidas técnicas e administrativas razoáveis para proteger os dados de navegação que possam ser coletados. Os dados são mantidos apenas pelo tempo necessário para cumprir as finalidades informadas, como análise de uso e melhoria contínua do serviço, ou conforme exigido por lei.

Como não armazenamos as imagens enviadas para conversão, não há retenção desses arquivos em nossos sistemas.
